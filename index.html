Projects

While much of my professional work is subject to confidentiality, the following projects illustrate the type of challenges I tackle and the solutions I create.

Project One - Standard tooling assemblies

Improve the tool design and assembly process by creating standard assemblies using relations in PTC Creo. This method drastically reduces the design and assembly time by creating standard fully constrained assemblies within seconds with little to no modifications required. Below is a GIF animation demonstrating the versatility of the final design. The animation shows the creation process of standard components with external data imported into the CAD software.

Project Two- Tooling improvement for high pressure frames

The clamp frames used for sealing are subjected to high pressure and cyclic loading during the forming process. This creates cracks on the sharp edges of the frame which propogates with continuous usage. This project aims explore possible ways to improve the existing frames with minimal changes to the existing design with the existing design constraints.
